Introduccion

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Se utiliza para estructurar el contenido en la web. Un documento HTML está compuesto por una serie de elementos, cada uno definido por etiquetas (tags). Estas etiquetas indican al navegador cómo mostrar el contenido.

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo que se utiliza para describir la presentación de un documento HTML. Mientras que HTML se encarga de la estructura del contenido, CSS se encarga de la apariencia visual. CSS permite definir estilos, como colores, fuentes, espaciamiento, y disposición del contenido.

HTML Basico
  • <!DOCTYPE html> Define el tipo de documento y la versión de HTML.
  • <html> Es la etiqueta raíz que envuelve todo el contenido de la página.
  • <head> Contiene metadatos sobre el documento, como el título.
  • <body> Contiene el contenido visible de la página web.
CSS Basico

Algunas de las propiedades mas utilizadas son:

  • background-color: Establece el color de fondo de un elemento.
  • font-size: Define el tamaño de la fuente.
  • font-weight: Define el grosor de la fuente
  • width: Define el ancho de un elemento.
  • margin: Define el margen exterior de un elemento.
  • padding: Define el relleno interior de un elemento.
  • display: Define cómo se muestra un elemento (por ejemplo, block, inline, flex, none)
  • position: Define cómo se posiciona un elemento (por ejemplo, static, relative, absolute, fixed).
Diseño y maquetacion

El diseño en CSS se refiere a la forma en que se estilizan y presentan los elementos HTML en una página web. CSS permite controlar aspectos visuales como colores, fuentes, márgenes, bordes, alineación, y mucho más

La maquetación en CSS se refiere a cómo se distribuyen y posicionan los elementos en una página web. Existen varias técnicas y herramientas en CSS para realizar la maquetación:

  1. Modelo de Caja: Cada elemento HTML se representa como una caja, que puede ser ajustada con propiedades como width, height, margin, padding, y border.
  2. Display
    • block: El elemento ocupa todo el ancho disponible.
    • inline: El elemento ocupa solo el espacio necesario y puede estar en línea con otros elementos.
    • flex: Utilizado para crear layouts flexibles y responsivos.
    • grid: Utilizado para crear layouts en rejilla.
  3. Posicionamiento
    • static: Posicionamiento por defecto, sigue el flujo normal del documento.
    • relative: Posiciona el elemento en relación a su posición original.
    • absolute: Posiciona el elemento en relación a su contenedor más cercano con posición relativa.
    • fixed: Posiciona el elemento en relación a la ventana del navegador.
    • sticky: Una mezcla entre relative y fixed.
  4. Flexbox: Un modelo de diseño unidimensional que facilita la distribución de espacio y alineación de elementos dentro de un contenedor. Propiedades clave incluyen flex-direction, justify-content, align-items, y flex-wrap.
  5. Grid: Un modelo de diseño bidimensional que permite crear layouts complejos con filas y columnas. Propiedades clave incluyen grid-template-columns, grid-template-rows, gap, grid-area, entre otras.
Multimedia en HTML

Propiedades de <audio>

  • src: Especifica la ruta del archivo multimedia. Generalmente se utiliza dentro de una etiqueta <source> para proporcionar múltiples fuentes.
  • controls: Añade controles de reproducción (play, pause, volumen) proporcionados por el navegador.
  • autoplay: Inicia la reproducción automáticamente cuando la página se carga. Nota: Esta propiedad puede ser deshabilitada por los navegadores debido a problemas de usabilidad.
  • loop: Reproduce el archivo multimedia en bucle, reiniciándose automáticamente cuando termina.
  • muted: Silencia el audio del archivo multimedia de manera predeterminada.
  • preload: Sugerencia al navegador sobre cómo manejar la precarga del archivo

Propiedades de <video>

  • width y height: Especifican las dimensiones del video en píxeles.
  • poster: Especifica una imagen que se muestra antes de que el video comience a reproducirse.
  • playsinline: Permite que el video se reproduzca en línea en dispositivos móviles en lugar de entrar en modo de pantalla completa.
Formularios en HTML
  • <form>
    • action: Especifica la URL a la que se enviarán los datos del formulario.
    • method: Especifica el método HTTP que se utilizará al enviar el formulario. Los valores comunes son GET y POST.
    • enctype: Especifica el tipo de contenido que se utilizará al enviar los datos del formulario. Comúnmente se usa multipart/form-data para subir archivos.
  • <input>
    • type: Especifica el tipo de entrada.
    • name: Especifica el nombre del campo de entrada, utilizado para identificar los datos cuando se envían
    • value: Especifica el valor predeterminado del campo de entrada.
    • placeholder: Especifica un texto de sugerencia que aparece dentro del campo de entrada cuando está vacío.
    • required: Indica que el campo de entrada debe ser completado antes de enviar el formulario.
    • readonly: Hace que el campo de entrada sea de solo lectura.
    • disabled: Deshabilita el campo de entrada, impidiendo que el usuario lo modifique.
    • maxlength: Especifica el número máximo de caracteres que se pueden ingresar.
    • min y max: Especifican los valores mínimo y máximo para campos de tipo numérico.
  • <textarea>
    • rows y cols: Especifican el número de filas y columnas del área de texto.
    • placeholder: Especifica un texto de sugerencia.
  • <select>
    • name: Especifica el nombre del menú desplegable.
    • multiple: Permite seleccionar múltiples opciones.
  • <button>
    • type: Especifica el tipo de botón. Los valores comunes son submit, reset, y button.
    • name y value: Especifican el nombre y valor del botón, utilizados al enviar el formulario.
